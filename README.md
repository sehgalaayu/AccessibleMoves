AccessibleMoves is a voice-activated navigation system designed to assist individuals with mobility challenges in navigating their environment seamlessly. Leveraging the power of Google Cloud's Speech-to-Text and Text-to-Speech services, the system translates voice commands into actionable navigation instructions.

Features

Voice-Activated Commands: Navigate through voice commands such as "start navigation", "turn left", "turn right", and "stop".
Real-Time Feedback: The system provides real-time feedback through voice responses for enhanced user experience.
Installation

Prerequisites
Python 3.11
Google Cloud account with Speech-to-Text and Text-to-Speech APIs enabled
Pyttsx3
SpeechRecognition
google-cloud-speech


Usage:

Run the main.py script:
bash
Copy code
